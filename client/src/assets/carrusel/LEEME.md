# Imágenes del carrusel de la página de inicio

Cada subcarpeta corresponde a una diapositiva del carrusel vertical:

- `rotulos/`      → Rótulos y letreros
- `senalizacion/` → Señalización
- `adhesivos/`    → Adhesivos y rotulación vehicular
- `acrilicos/`    → Acrílicos
- `publicidad/`   → Publicidad a la medida

Suelte aquí las fotos (`.jpg`, `.jpeg`, `.png` o `.webp`) de cada categoría.
El carrusel las toma **automáticamente**: si una carpeta tiene varias imágenes,
la diapositiva las rota con un cross-fade. Si una carpeta está vacía, se muestra
la ilustración vectorial de la marca como respaldo.

El orden de aparición es alfabético por nombre de archivo, así que puede
nombrarlas `01-...`, `02-...`, etc. para controlar la secuencia.
